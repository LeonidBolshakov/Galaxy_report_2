# Galaxy Report

Настольная утилита для подготовки финансовых значений к ежемесячному отчёту
партнёра в корпорацию «Галактика».

## Документация

- [Русская версия](README.ru.md)
- [English version](README.en.md)
- [Сборка exe](docs/build.md)
- [Архитектура проекта](docs/architecture.md)

Кратко: приложение принимает сумму, заплаченную клиентами, и платежи в
корпорацию; рассчитывает значения для отчёта; каждое из шести полей результата
можно скопировать в буфер обмена кликом по этому полю.
