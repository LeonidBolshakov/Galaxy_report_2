# Galaxy Report

Настольная утилита для подготовки финансовых значений к ежемесячному отчёту
партнёра в корпорацию «Галактика».

Приложение принимает сумму, заплаченную клиентами, и платежи в корпорацию.
НДС рассчитывается по проценту НДС. Процент перечисления в корпорацию и процент
НДС задаются константами программы, но могут быть исправлены в интерфейсе на
период текущей сессии.

## Возможности

- Настольный интерфейс на PyQt6, созданный на основе файла Qt Creator `.ui`.
- Автоматический расчёт:
  - суммы клиентов без НДС;
  - суммы перечисления в корпорацию без НДС;
  - суммы перечисления в корпорацию с НДС;
  - общей суммы платежей в корпорацию;
  - остатка к оплате;
  - переплаты.
- Динамические строки ввода платежей: после суммы платежа появляется следующее
  поле.
- Завершить ввод можно клавишей `Tab`, клавишей `Enter` или переходом мышью;
  `Enter` переводит фокус как `Tab`.
- Клик по любому полю результата копирует в буфер обмена текст результата.
- Скопированные поля результата подсвечиваются.
- Поля результата растягиваются вместе с формой, поэтому длинные значения проще
  просмотреть перед копированием.
- Справочные значения — процент НДС и процент перечисления в корпорацию — можно
  изменить только на текущую сессию после явного подтверждения.
- Проект покрыт pytest-тестами: расчёты, форматирование, поведение интерфейса,
  динамические платежи, копирование в буфер обмена и защита справочных значений.

## Использование

1. Ввести сумму, заплаченную клиентами.
2. Поочерёдно ввести платежи в корпорацию.
3. После суммы платежа появляется следующее поле.
4. После ввода всех платежей перенести рассчитанные результаты в ежемесячный
   отчёт в корпорацию.
5. Клик по любому полю результата копирует в буфер обмена текст результата для
   вставки в ежемесячный отчёт в корпорацию.
6. При необходимости изменить процент перечисления в корпорацию или процент НДС
   на текущую сессию.

## Правила расчёта

- Все денежные значения округляются до копеек.
- Процент НДС и процент перечисления в корпорацию задаются константами.
- Остаток и переплата взаимоисключающие: если есть переплата, остаток равен
  нулю, и наоборот.
- После перезапуска приложения справочные значения восстанавливаются из констант.

```text

├── constants.py             # Константы приложения
├── functions.py             # Форматирование, парсинг и буфер обмена
├── main.py                  # Точка входа приложения
├── report.py                # Главное окно и бизнес-сценарий
```

## Установка

Создать виртуальное окружение и установить зависимости:

```powershell
python -m venv .venv
.\.venv\Scripts\python.exe -m pip install -r requirements.txt
```

## Запуск

```powershell
.\.venv\Scripts\python.exe main.py
```

## Тесты

```powershell
.\.venv\Scripts\python.exe -m pytest .
```

Последний проверенный результат: `17 passed`.

## Сборка exe

```powershell
.\build_pyinstaller.bat
```

Готовый exe создаётся здесь:

```text
dist\galaxy_report\galaxy_report.exe
```
